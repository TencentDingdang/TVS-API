# 二. 功能介绍

用户进行语音交互时，云小微向设备下发不同的表情、动作标签。设备可以自由定义不同的行为。

# 三. 功能流程

示例：
- 用户发起交互说：我很喜欢你
- 云小微下发“喜欢”的表情数据和“欢迎”的动作数据
- 设备根据表情数据在屏幕上显示一个“喜欢”动画表情，根据“欢迎”的动作数据做出欢迎的动作。
